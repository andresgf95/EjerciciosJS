Apuntamentos

Os strings en JS teñen un bo número de métodos que permiten facer moitas cousas.

Un dos métodos é .toUpperCase

const texto = "Hola, que tal"
const textoMaiusculas = texto.toUpperCase()
// textoMaiusculas é "HOLA, QUE TAL"

Exercicio

Imos facer unha aplicación que transcriba en maiusculas o texto que escribamos.

  1. Captura o textarea nunha constante entradaDoTexto.
  2. Captura o p con id="enMaiusculas" nunha constante saidaDoResultado.
  3. Define unha función chamada actualizaSaidaEnMaiusculas
    Será algo como function actualizaSaidaEnMaiusculas() {}
  4. No bloque da funcion {}, define o seguinte proceso
    1. Asignamos á constante textoIntroducido o valor do atributo entradaDoTexto.value
    2. Asignamos á constante textoEnMaiusculas o valor de textoIntroducido.toUpperCase().
    3. Asignamos á propiedade saidaDoResultado.innerText o valor de textoEnMaiusculas
  5. Vincula a función actualizaSaidaEnMaiusculas ó evento "input" de entradaDoTexto.
    Faino empregando entradaDoTexto.addEventListener.

Escribe o código no ficheiro ./resposta.js

Consulta a consola de JS do navegador para ver os erros que se podan estar a producir.
