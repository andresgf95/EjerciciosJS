Apuntamentos

Con frecuencia, é necesario aplicar múltiples procesos á información para chegar ó resultado que procuramos.

Para este exercicio necesitaremos coñecer cántos caracteres ten un string, pero tamén cantas palábras.

Os string en JavaScript teñen unha propiedade .length que nos dí o número de caracteres que ten o string.

Pero JavaScript non mañexa o concepto de "palabras" así que teremos que pensar cómo conseguir saber o número de palabras no string.

Deixando a un lado a informática ¿Cómo sabes tí cantas palabras ten un texto? ¿Cómo sabes onde empeza e remata unha palabra?.

Os strings dispoñen dun método que nos permite trocear o string en pedaciños, partindo por cada sitio onde se atope o caracter que indiquemos.

Cando o empregamos, obtemos un array no que cada elemento é un trociños do string orixinal.

const textoInicial = "Ola, que tal"
const textoDivididoPorEspacios = textoInicial.split(" ")
// textoDivididoPorEspacios conten ["Ola,","que","tal"]

Os array tamén dispoñen da propiedade .length, así que é dodado saber cantos elementos hai nun array.

¿Ves xa o camiño para saber o número de palabras nun string?

Imos contar as letras e palábras mentras se escriben nun elemento textarea.

Escribe o código no ficheiro ./resposta.js

    2. Engade un manexador para o evento "input" no textarea para que execute a función actualizaNumeroPalabras.
